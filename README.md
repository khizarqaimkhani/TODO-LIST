# 📝 Simple To-Do List

A minimal and responsive **To-Do List App** built using **HTML, CSS, and JavaScript**.  
It helps you manage daily tasks easily with options to add, mark, and delete items.

---

## ⚙️ Features
- ➕ Add new tasks  
- ✅ Mark tasks as completed  
- ❌ Delete tasks  

---

## 💻 Technologies Used
- 🌐 HTML  
- 🎨 CSS  
- ⚡ JavaScript
